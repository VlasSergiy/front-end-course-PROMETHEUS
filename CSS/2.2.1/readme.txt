Стилізуйте HTML елементи відповідно до наступних правил:
1. Заголовок першого рівня: колір тексту зелений (використовуйте ключове слово green)
2. Заголовок другого рівня: колір тексту пурпурний (використовуйте ключове слово magenta)
3. Параграф: колір тексту білий, а колір фону чорний (використовуйте відповідні ключові слова)
4. Кнопка: колір тексту червоний (використовуйте ключове слово red), а колір фону рожевий (використовуйте ключове слово rosybrown)
