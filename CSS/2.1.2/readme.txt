Опис завдання:
Виконайте завдання за допомогою CSS стилів.

Для всіх посилань (тегів <a>) задайде наступні стилі:
- приберіть підкреслення
- задайте чорний колір тексту (за допомогою ключового слова black)
- встановіть розмір шрифту 18px
Окремо виберіть посилання з текстом "Головна" за допомогою прямого дочірнього селектора ( знаку ">" між батьківським і дочірнім елементами в селекторі) та зробіть текст червоного кольору (колір задавати за допомогою ключового слова red)

Зразок:

Для виконання завдання скопіюйте шаблон в робочу область та замініть коментарі на відповідні стилі:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>CSS селектори</title>
    <style>
      a {
        font-size: 18px;
  
        /* додайте CSS стилі замість коментаря */

      }
  
      /* додайте CSS стилі замість коментаря */
  
    </style>
  </head>
  
  <body>
    <nav>
      <a href="#">Головна</a>
      <ul>
        <li><a href="#">Про нас</a></li>
        <li><a href="#">Наші роботи</a></li>
        <li><a href="#">Відгуки</a></li>
        <li><a href="#">Контакти</a></li>
      </ul>
    </nav>
  </body>
</html>
