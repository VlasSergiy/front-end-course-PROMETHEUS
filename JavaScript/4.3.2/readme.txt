Опис завдання

Маємо вкладений масив, що репрезентує сидіння в літаку:

const airplaneSeats = [
    ["Anna", "Rob", "Stevie"],
    ["Luisa", "Pedro", "Maya"],
    ["Luli", "Olena", "Iryna"],
    ["Marta", null, "Deniz"],
    ["Rin", "Ryan", "Francisco"]
];

В літаку одне місце вільне і позначено як null. Забронюйте це місце для пасажира на ім”я "Robert" (оновіть елемент, який зараз має значення null так, щоб воно перетворилось на "Robert").
Жодних нових змінних оголошувати не потрібно, працюйте з airplaneSeats.
Перевіряються всі змінні на коректність присвоєних значень.
