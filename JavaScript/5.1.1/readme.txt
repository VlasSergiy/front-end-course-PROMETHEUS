Опис завдання

Оголосіть новий об'єкт з назвою cat. Це має бути об'єктний літерал з наступними даними в межах літералу:

- властивості nickname присвоїти значення Tom (тип String)
- властивості age присвоїти значення 3 (тип Number)
- властивості isBlack присвоїти значення true (тип Boolean)
- властивості games присвоїти масив ["jumping", "running", "making mess"]
