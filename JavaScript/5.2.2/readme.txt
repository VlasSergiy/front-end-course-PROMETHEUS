Опис завдання

Реалізуйте функцію, що називається getLastElement. Функція прийматиме масив і повертатиме його останній елемент. Якщо масив порожній, то повертатиме null.

Приклад викликів:

console.log(getLastElement([ 1, 2, 3 ])); //  3
console.log(getLastElement([ 'one', 'two'])); //   'two'
console.log(getLastElement([ true ])); //   true
console.log(getLastElement([])); //  null
