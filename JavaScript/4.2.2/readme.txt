Опис завдання

Маємо оголошену змінну:

const word = "usually";

Застосовуючи вбудовані методи для даних типу String щодо змінної word, перетворіть її значення у “sea”.
- використовуючи метод slice, виокремте частину “sua”
- використовуючи метод replace, замініть “u” на “e”
- створіть ланцюжок методів із перелічених
- збережіть результат у змінну result
Перевіряються всі змінні на коректність присвоєних значень, тому НЕ змінюйте змінну word.
