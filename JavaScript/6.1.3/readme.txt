Опис завдання
Використовуючи JavaScript зробіть наступні зміни:
- зверніться до <div> за id (getElementById) та задайте йому розміщення тексту посередині
- зверніться до зображення за назвою тега використовуючи querySelector та задайте ширину 150px та заокруглення кутів 50%
- у всіх випадках для роботи зі стилями використовуйте об”єкт style (наприклад: some_element.style.fontSize = "30px")
Шаблон в робочу область копіювати НЕ потрібно. Додайте тільки реалізацію.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>DOM</title>
  </head>
  
  <body>
    <div id="container">
      <h1>I love Trees</h1>
      <img src="https://image" alt="tree" />
    </div>
  </body>
</html>
