Опис завдання

Створіть функцію з назвою getDayOfWeek, що приймає один параметр - число від 1 до 7 - та повертає день тижня англійською (якщо 1 - то Monday, якщо 2 - то Tuesday і так далі) або null (undefined теж допускається), якщо передане число менше 1 чи більше 7.

Приклади викликів:

console.log(getDayOfWeek(1)); // -> "Monday"
console.log(getDayOfWeek(7)); // -> "Sunday"
console.log(getDayOfWeek(0)); // -> null
