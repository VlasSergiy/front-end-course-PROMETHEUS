Опис завдання

Маємо масив, що складається з планет:

let planets = ['The Moon', 'Venus', 'Earth', 'Mars', 'Jupiter'];
Використовуючи методи масиву pop/push/shift/unshift
- видаліть перший елемент 'The Moon'
- в кінець масиву додайте елемент 'Saturn'
- першим елементом масиву додайте 'Mercury'
- жодних нових змінних оголошувати не потрібно, працюйте з planets
Перевіряються всі змінні на коректність присвоєних значень.
