Опис завдання

За допомогою fetch API потрібно надіслати запит для створення нової статті.

- змінній url присвоєно ендпоїнт, на який потрібно надіслати дані
- змінній article присвоємо об'єкт статті, яку потрібно надіслати на сервер
Налаштуйте fetch запит, використовуючи оголошені змінні.

Для виконання завдання скопіюйте шаблон в робочу область та замініть коментар на Вашу реалізацію:

const url = "https://jsonplaceholder.typicode.com/posts";

const article = {
   title: "New Title",
   content: "New Content"
};
