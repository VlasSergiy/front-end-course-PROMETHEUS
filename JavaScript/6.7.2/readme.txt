Опис завдання
Використовуючи addEventListener, додайте слухач події клік двом кнопкам, звертаючись до них за id (getElementById)

- коли, користувач натисне кнопку з id "hello" - у консоль вивести "hello"
- коли, користувач натисне кнопку з id "goodbye" - у консоль вивести "goodbye"
Шаблон в робочу область копіювати НЕ потрібно. Додайте тільки реалізацію.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Events</title>
  </head>
  
  <body>
    <button id="hello">Hello</button>
    <button id="goodbye">Goodbye</button>
  </body>
</html>
