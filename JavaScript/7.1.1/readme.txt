Опис завдання

Ви керівник команди, яка розробляє гру "Хом'якова ферма". Один із розробників отримав завдання створити функцію-конструктор "хом'як" (англ. - "Hamster").

Об'єкти-хом'яки повинні мати масив food для зберігання їжі та метод foundFood, який додає до масиву все, що знайшов хом'як.

Нижче його рішення. При створенні двох хом'яків, якщо поїв один — чомусь ситим стає другий теж.

В чому справа? Як виправити?

Приклад використання коду:

function Hamster() {}

Hamster.prototype.food = []; // => порожній "шлунок"

Hamster.prototype.foundFood = function(something) {
    this.food.push(something);
};

// Створюємо двох хом'яків 
let speedy = new Hamster();
let lazy = new Hamster();

// і годуємо тільки першого
speedy.foundFood("apple");
speedy.foundFood("nut");

console.log(speedy.food.length); // 2
console.log(lazy.food.length);   // 2  => чому другий хом'як також отримав елементи у масив food і як це виправити?

Виправіть помилку у функціоналі, що описано в робочій області.
