Опис завдання

Маємо оголошену змінну*:

let message = "      I love JavaScript!!!      ";

*майте на увазі, що не варто покладатись на наведене у прикладі значення змінної. Ваше рішення має працювати для будь-якого значення типу String, присвоєного змінній message.

Оголосіть змінну yell, що має містити значення змінної message, але у верхньому регістрі та без зайвих пробілів на початку і вкінці.

Для виконання завдання використайте вбудовані методи для даних типу String та створіть з них ланцюжок методів.

Перевіряються всі змінні на коректність присвоєних значень, тому НЕ змінюйте змінну message.
Редактор коду
let message = "      I love JavaScript!!!      ";
let yell = (message.trim().toUpperCase());
