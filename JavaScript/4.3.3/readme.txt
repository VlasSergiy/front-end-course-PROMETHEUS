Опис завдання

Маємо масив чисел:

let numbers = [4, 10, 23, 100, 80, 35, 0, 11];
Не змінюючи початковий масив та використовуючи методи масиву:
- відфільтруйте елементи масиву, значення яких більше 10
- перетворіть результуючий масив на масив об'єктів, де кожен елемент буде присвоєно параметру num (наприклад: { num: 23 })
- використовуйте ланцюжок методів
- результат присвойте змінній result
Очікуваний результат:

let result = [
  { num: 23 },
  { num: 100 },
  { num: 80 },
  { num: 35 },
  { num: 11 }
];
Перевіряються всі змінні на коректність присвоєних значень.
