Опис завдання
Зверніться до тега <input> за допомогою querySelector та присвойте його змінній input. Задайте йому атрибут value зі значенням "Default text" та атрибут id зi значенням "my-input".

Шаблон в робочу область копіювати НЕ потрібно. Додайте тільки реалізацію.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>DOM</title>
  </head>
  
  <body>
    <input type="text" />
  </body>
</html>
