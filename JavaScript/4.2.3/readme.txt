Опис завдання

Маємо оголошену змінну*:

const str = "hello";

Звертаючись до кожного символу рядка за його індексом, перетворіть значення "hello" у “olleh”.
Збережіть результат у змінну reverseString.
*майте на увазі, що не варто покладатись на наведене у прикладі значення змінної. Ваше рішення має працювати для будь-якого значення типу String, що складається з п'яти символів, присвоєного змінній str.
Наприклад: const str = "qwert"; має бути перетворено у "trewq".

Перевіряються всі змінні на коректність присвоєних значень, тому НЕ змінюйте змінну str.
