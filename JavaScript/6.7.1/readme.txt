Опис завдання
Відразу наголошу, що я не рекомендую додавати слухачі подій прямо у HTML код, але для розігріву почнемо з них.

- елементу <h1> додайте лінійний слухач події клік, та обробник події, який буде виводити в консоль "I do not like inline events…"
- елементу <button> додайте лінійний слухач події клік, та обробник події, який буде виводити в консоль "Clicked!"
- реалізуйте все повністю в межах HTML елементів. Додаткових функцій/змінних/тегів оголошувати не потрібно
Для виконання завдання скопіюйте шаблон в робочу область та замініть коментарі на Вашу реалізацію:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Events</title>
  </head>
  
  <body>
    <h1 /*замініть коментар на Вашу реалізацію */ >I do not like inline events…</h1>
    <button id="btn" /*замініть коментар на Вашу реалізацію */ >Click Me!</button> 
  </body>
</html>
